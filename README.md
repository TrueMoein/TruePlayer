# TruePlayer
Advanced Player

Install:

`npm i --save trueplayer`

or

`yarn add trueplayer`

How to use:

```javascript

import TruePlayer from 'trueplayer';

const player = new TruePlayer('my-video'); // select video with my-video id

player.seek(10); // go to video seconds of 10
player.play(); // play the video

```
